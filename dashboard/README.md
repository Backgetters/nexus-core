# NEXUS Mission Control Dashboard

## Quick Start

```bash
# Start the dashboard server
./start-dashboard.sh

# Or manually with Python
cd ~/clawd/dashboard && python3 -m http.server 8080
```

Then open: http://localhost:8080

## Features

- **Overview**: System status, quick stats, live terminal
- **File Browser**: Quick access to key directories
- **API Status**: Real-time connection status for all APIs
- **System Logs**: Recent activity and events
- **Earnings**: ClawWork revenue tracking
- **Memory Bank**: NEXUS memory system access

## Dashboard Sections

1. **System Status**: Disk, memory, agents, last check
2. **Quick Stats**: Files, size, skills, memory snapshots
3. **Monitoring**: Heartbeat schedule, health, alerts
4. **Live Terminal**: Real-time system messages
