# GitHub Skill

Interact with GitHub using the `gh` CLI for repository management, PR handling, and Git operations.

## Installation

Requires `gh` CLI installed:
```bash
brew install gh  # macOS
```

## Usage

- Repository operations
- Pull request management
- Issue tracking
- Git workflow automation

## Examples

- `gh repo view` - Show repository info
- `gh pr list` - List pull requests
- `gh issue create` - Create issues

This skill enables comprehensive GitHub integration for your OpenClaw agent.
