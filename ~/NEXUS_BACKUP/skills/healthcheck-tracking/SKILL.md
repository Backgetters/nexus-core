# HealthCheck Tracking System

Track water intake and sleep patterns with JSON file storage for health monitoring.

## Features

- Water intake tracking
- Sleep pattern monitoring
- JSON-based data storage
- Health trend analysis
- Goal setting and achievement

## Usage

Monitor health metrics with:
- Daily water intake tracking and goals
- Sleep pattern analysis and optimization
- JSON-based data persistence
- Health trend identification
- Achievement tracking and motivation

Essential for maintaining personal health and wellness tracking.
