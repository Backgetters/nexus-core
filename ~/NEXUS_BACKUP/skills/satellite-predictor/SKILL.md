# Satellite Pass Predictor

Predict satellite passes (NOAA APT, METEOR LRPT, ISS) for configured locations with real-time tracking.

## Features

- Satellite pass prediction
- Multiple satellite support (NOAA, METEOR, ISS)
- Location-based predictions
- Real-time tracking updates
- Pass visibility calculations

## Usage

Track satellites with:
- Accurate satellite pass predictions
- Multiple satellite system support
- Location-based visibility calculations
- Real-time tracking and updates
- Comprehensive pass scheduling

Perfect for satellite communication, weather data reception, and space station tracking.
