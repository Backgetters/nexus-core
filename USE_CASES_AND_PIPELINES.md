# NEXUS Use Cases & Pipelines

## Battle-Tested Agent Workflows

### Pipeline 1: Morning Intelligence Briefing
**Skills:** blogwatcher + weather + email-triager + apollo

```
1. blogwatcher → Check RSS feeds for industry news
2. apollo → Enrich key contacts mentioned in news
3. email-triager → Prioritize overnight emails
4. weather → Check conditions for meetings
5. Output: Consolidated morning briefing
```

### Pipeline 2: Content Creation Engine
**Skills:** algorithmic-art + cold-email + seo-content-engine + social-media-content

```
1. algorithmic-art → Generate hero images
2. seo-content-engine → Write optimized blog post
3. cold-email → Create outreach campaign
4. social-media-content → Generate social posts
5. Output: Complete content package
```

### Pipeline 3: Code Review & Deployment
**Skills:** agentlens + github + aws-ecs-monitor + debug-pro

```
1. agentlens → Navigate and understand codebase
2. github → Check PR status, CI runs
3. debug-pro → Systematic debugging if issues
4. aws-ecs-monitor → Verify deployment health
5. Output: Code review + deployment status
```

### Pipeline 4: Sales Automation
**Skills:** apollo + cold-email + lead-scorer + crm-manager

```
1. apollo → Find and enrich prospects
2. lead-scorer → Prioritize high-value leads
3. cold-email → Write personalized outreach
4. crm-manager → Update CRM with new contacts
5. Output: Qualified leads + outreach sent
```

### Pipeline 5: Personal Knowledge Management
**Skills:** notion + obsidian + doc-coauthoring + essence-distiller

```
1. essence-distiller → Extract key ideas from content
2. obsidian → Create linked notes
3. notion → Organize into databases
4. doc-coauthoring → Collaborate on documents
5. Output: Organized knowledge base
```

### Pipeline 7: Startup Operations
