# Terraform Cloud Skill

Manage Terraform Cloud workspaces, runs, and state.

## Usage

- `terraform-cloud workspaces list` - List workspaces
- `terraform-cloud runs list` - List recent runs
- `terraform-cloud apply <run_id>` - Apply run
- `terraform-cloud state pull` - Pull state file

## Requirements

- Terraform CLI
- TFE_TOKEN environment variable
