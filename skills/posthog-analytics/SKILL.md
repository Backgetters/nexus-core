# PostHog Analytics Skill

Product analytics, feature flags, and session replay with PostHog.

## Usage

- `posthog events list` - List recent events
- `posthog insights list` - List saved insights
- `posthog featureflags list` - List feature flags
- `posthog cohorts list` - List user cohorts

## Requirements

- POSTHOG_API_KEY and POSTHOG_PROJECT_ID environment variables
