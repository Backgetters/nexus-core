# Grafana Dashboards Skill

Manage Grafana dashboards, alerts, and data sources.

## Usage

- `grafana dashboards list` - List dashboards
- `grafana dashboard get <uid>` - Get dashboard JSON
- `grafana alerts list` - List alert rules
- `grafana datasource add` - Add data source

## Requirements

- GRAFANA_URL and GRAFANA_API_KEY environment variables
